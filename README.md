# MyBlog HTML Project

Educational project for practicing semantic HTML.

## Features

- Semantic layout (header, nav, section, article, aside, footer)
- Accessible form with validation
- SEO meta description
- Figure and time elements

## Technologies

- HTML5

## What I learned

- Semantic structure
- Accessibility basics
- Form validation
- SEO fundamentals

## How to run

Open index.html in your browser.
